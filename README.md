# Capstone Project
Visitors Guide for The Magic City Discovery Center (a local children's museum) an iOS Appplication

## Background: 
This capstone was to encompose the full lifecycle of mobile app developement. I spent a good deal of time learning UI/UX developement tools including Figma, to help showcase prototypes as well as develop well commented documentation for the developement process. In this there will be links and screenshots of the intitial developemental phases, design processes, and developement of this project. Again, the overall intention was to get hands on practice with project planning, UI/UX developement, and full-stack development.

### Tech Stack
- Swift
- SwiftUI / UIkit
- Firebase
